How to create a Repository on GitHub?

Step-1: Create account on GitHub.com

Step-2: Click on New Repository

Step-2: Write Repository Name in field as below(For example: My Repository Name is Guide-on-VCS-usingGitHub):

Step-3: Select your Repository Type, either public or private and check box if you want to add ReadME file. Now click on create Repository

Step-4: Congratulations!! Your Repository has been created!

How to Commit & Push local repository on GitHub?

Step-1: git clone /path/to/repository | Command is used to Create a working copy of a remote GitHub repository

Step-2: git init | Command is used to initialize a folder into Git Repository

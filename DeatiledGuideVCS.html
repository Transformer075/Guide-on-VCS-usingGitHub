How to create a Repository on GitHub?

Step-1: Create account on GitHub.com

Step-2: Click on New Repository

Step-2: Write Repository Name in field as below(For example: My Repository Name is Guide-on-VCS-usingGitHub):

Step-3: Select your Repository Type, either public or private and check box if you want to add ReadME file. Now click on create Repository

Step-4: Congratulations!! Your Repository has been created!

How to Commit & Push local repository on GitHub?

Step-1: git clone /path/to/repository | Command is used to Create a working copy of a remote GitHub repository

Step-2: git init | Command is used to initialize a folder into Git Repository

Step-3: git add | Command is used to add a single file to your commit (It will not directly commit, it will just add file into stagging area, an intermediate area between local & GitHub Repository

Step-4: git commit -m "Commit Message" | Command is used to Commit changes to GitHub Repository (but not yet to the remote repository):

Step-5: git push | Command is used to Push changes to GitHub server. These changes will be available to your teammates.

Useful Git Commands are as follows:

git status | Command is used to List the files you've changed and those you still need to add or commit

git branch | Command is used to List all branches

git pull | Command is used to get latest Updates from your GitHub Repository where your friends have updated the source code
